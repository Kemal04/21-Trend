Return
How can I return the products I bought?

Return steps:

1. Go to My Orders from the My Account section of trendyol.com and mobile applications.

2. Press the Details button and view the details of your order.

3. Click the "Create Easy Return Request" button.

4. Select the item to be returned and the reason for the return. If you have purchased more than one product of the same product, you must also select the number of products to be returned.

5. Make your shipping selection.

6. Take note of the return shipping code that appears on the screen. You can also access your return shipping code from the "My Orders" page and the notification message sent to your membership e-mail address.

7. Put the invoice together with the items to be returned in one package. (The return code must be obtained separately for each delivery and the packages must be prepared and shipped separately.)

If you do not have an invoice, you can write the following information on a blank piece of paper and put it in the return package and return it.

o Name Surname:

o Order No:

o Reason for Return:

8. Deliver the package with the return shipping code to your chosen courier within 7 days. Simply provide the code, and you don't need to specify a Returns address. In case you pass 7 days, you should get a return shipping code again.
